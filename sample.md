# Welcome to Markdown Reader

Drop a Markdown file anywhere in this window, or choose **File → Open**.

## Included features

- Live rendered preview
- Source editing
- Save and Save As
- PDF export
- Zoom controls
- Find, undo, redo, and common keyboard shortcuts

> Your local images and links are resolved relative to the open Markdown file.

| Shortcut | Action |
|---|---|
| `Ctrl+O` | Open |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+P` | Export PDF |
| `Ctrl+F` | Find |
| `Ctrl++` / `Ctrl+-` | Zoom |
